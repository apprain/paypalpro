
Step 1:
    Method 1: 
        1.	First Login to admin panel 
        2.	Go to : (Preference) > (Configuration) 
            Set, Debug Mode = 1
        3.	Click on (Application) tab
        4.	Click on (Install New Component)
        5.	Select zip file
        6.	Click on Install Button. 
        7.	After upload, this extensions will appear in the list.
            Now click on "Activate" link blow description. 
            
    Method 2: 
        1.	First Login to admin panel 
        2.	Go to : (Preference) > (Configuration) 
            Set, Debug Mode = 1
        3.	Unzip extensions file(zip).
        4.	Copy the folder in "component" directory in your project
        5.	In Admin panel go to application tab and find the component
        6.	Click on Activate Link. 
Step 2:
    Go   After activation to appSore and payment method section. Newly installed method will be shown there, activate it.
